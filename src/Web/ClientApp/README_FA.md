# سامانه پایش هوشمند نرخ رمزارزها

یک اپلیکیشن مدرن، بسیار منسجم و حرفه‌ای برای مانیتورینگ زنده نرخ ارزهای دیجیتال. این پروژه شامل یک فرانت‌اند پیشرفته بر پایه **آنگولار ۲۱ (نسخه بدون Zone.js مجهز به سیگنال‌های واکنشی)** و استایلینگ زیبای **Tailwind CSS v4** است که به یک بک‌اند وب اکسپرس (Express.js) یکپارچه‌سازی شده با هوش مصنوعی مدل **Gemini 3.5-Flash** متصل گردیده تا نرخ‌های زنده را با ابزار جستجوی گوگل (Google Search Grounding) دریافت و تحلیل نماید.

---

## ✨ ویژگی‌های برجسته پروژه

- **معماری بدون زون‌جی‌اس (Zoneless)**: پیاده‌سازی شده مطابق با آخرین استانداردهای سال ۲۰۲۶ آنگولار جهت تضمین حداکثر کارایی (Peak Performance) و سبک‌سازی رندرسازی با استفاده از سیگنال‌های پیشرفته.
- **راست‌چین و چپ‌چین جادویی (RTL & LTR)**: مجهز به سیستم بومی تغییر بی‌درنگ زبان بین فارسی (راست‌چین) و انگلیسی (چپ‌چین) در زمان اجرا (Runtime).
- **تم‌های منعطف تیره و روشن**: تم تاریک اختصاصی با افکت‌های نوری و گرادیانت‌های مدرن در کنار تم روشن مینیمال، چشم‌نواز و استاندارد.
- **پایش لحظه‌ای با ابزار جستجو (Search Grounding)**: استخراج داده‌های بهینه با کتابخانه رسمی `@google/genai` در سمت بک‌اند اکسپرس با قابلیت ساختارسازی خروجی به شکل دقیق‌ترین قالب JSON.
- **مکانیزم بازیابی و پایداری بالا (High-Fidelity Fallbacks)**: مجهز به داده‌های شبیه‌سازی‌شده پایدار و محاسباتی پویا جهت نمایش بدون نقص المان‌ها و کارت‌های نرخ ارز در زمان قطعی اینترنت یا اعمال محدودیت سهمیه وب ارائه‌دهنده.
- **رابط کاربری کاملاً ریسپانسیو**: طراحی شده برای تمام دستگاه‌ها (موبایل، تبلت و دسکتاپ‌های عریض) همراه با دکمه‌های مناسب جهت لمس در نمایشگرهای لمسی و انیمیشن‌های ملایم وضعیت‌ها.

## 🛠️ معماری و فناوری‌های مورد استفاده

### فرانت‌اند (Frontend)
- **فریم‌ورک اصلی**: آنگولار ۲۱ (شامل اجزای Standalone بدون وابستگی به زون‌جی‌اس)
- **مدیریت وضعیت**: Angular Signals همراه با سیگنال‌های محاسباتی `computed()`
- **استایل‌دهی**: Tailwind CSS v4 (ادغام‌شده با `@tailwindcss/postcss` برای کامپایل سریع‌تر)
- **آیکون‌ها**: گنجینه استاندارد متریال آیکون آنگولار (`<mat-icon>`)

### بک‌اند (Backend)
- **فریم‌ورک سرور مانیتورینگ**: Express.js
- **هسته‌ هوش مصنوعی**: Google Gemini 3.5-Flash با قابلیت ابزار جستجوگر وب گوگل (Google Search Tools)
- **سیستم مدیریت و سرویس**: ساختاربندی هیبریدی SSR جهت تحویل کدهای بهینه‌سازی‌شده فرانت‌اند

## 📂 ساختار دایرکتوری‌های پروژه

```text
├── angular.json          # پیکربندی معماری کلاینت آنگولار
├── package.json          # ثبت وابستگی‌های پروژه و اسکریپت‌های اجرایی
├── .env.example          # فایل نمایشی الگو برای متغیرهای محیطی
├── src/
│   ├── main.ts           # نقطه آغازین اجرای کلاینت در مرورگر
│   ├── main.server.ts    # نقطه راه‌اندازی رندر سمت سرور (SSR)
│   ├── server.ts         # سرور اکسپرس بک‌اند (طراحی API نرخ‌ها با Gemini و سیستم سقوط ایمن)
│   ├── styles.css        # شیوه نامه اصلی استایل پروژه با ایمپورت‌های تیلویند ۴
│   └── app/
│       ├── app.ts        # کنترل‌کننده کلاینت و سیگنال‌ها
│       ├── app.html      # قالب ساخت یافته کارت‌های نرخ رمزارز
│       └── translations.ts # واژه‌نامه محلی زبان انگلیسی و فارسی
└── public/               # پوشه توزیع دارایی‌های ثابت (Assets)
```

## ⚙️ راهنمای نصب و اجرای پروژه

برای راه‌اندازی و اجرای این محصول بر روی سیستم محلی خود، گام‌های زیر را دنبال کنید:

### پیش‌نیازها
- **Node.js**: نسخه `v20.x` یا بالاتر
- **npm**: نسخه `v10.x` یا بالاتر

### ۱. تنظیم متغیرهای محیطی
یک فایل با نام `.env` در مسیر اصلی پروژه خود بسازید:
```bash
cp .env.example .env
```
سپس کلید خصوصی درگاه جمینی خود را در این فایل بنویسید:
```env
GEMINI_API_KEY=your_actual_gemini_api_key_here
```

### ۲. نصب پکیج‌های پیش‌نیاز
دستور زیر را برای دانلود و نصب وابستگی‌ها اجرا کنید:
```bash
npm install
```

### ۳. اجرای محیط توسعه کلاینت و سرور
جهت آغاز محیط مانیتورینگ و توسعه محلی روی پورت `3000` دستور زیر را وارد نمایید:
```bash
npm run dev
```
اکنون آدرس [http://localhost:3000](http://localhost:3000) را در مرورگر خود باز کنید.

### ۴. ساخت نهایی پروژه (Build)
برای گرفتن خروجی نهایی بهینه شده و کامپایل شده بک‌اند و فرانت‌اند برای استقرار (Deployment):
```bash
npm run build
```
سپس برای میزبانی رسمی محصول روی سرور اصلی از این دستور استفاده کنید:
```bash
npm run serve:ssr:app
```
